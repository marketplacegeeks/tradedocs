## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Unauthenticated → API | JWT layer already rejects unauthenticated callers globally (DEFAULT_PERMISSION_CLASSES = IsAuthenticated) |
| Authenticated Checker → write endpoints | Checkers are authenticated but must not reach create/update/destroy paths |
| Document creator → APPROVE action | Creator is authenticated and holds a valid role, but must not approve their own documents |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-3-01 | Elevation of Privilege | `IsMakerOrAdmin` class | mitigate | Class enforces MAKER/COMPANY_ADMIN/SUPER_ADMIN only — Checker denied at routing layer before `perform_*` is called |
| T-3-02 | Denial of Service | Document creation endpoints | mitigate | ScopedRateThrottle 100/day per user blocks scripted document flooding |
| T-3-03 | Elevation of Privilege | `WorkflowService.transition()` APPROVE action | mitigate | Self-approval guard raises PermissionDenied when created_by == performed_by and role is not COMPANY_ADMIN/SUPER_ADMIN |
</threat_model>
